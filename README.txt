FTP Client Project - For CSCI 355

This is basically going to be a simple FTP client that will be as good as we can manage by the due date.
We're using Git to organize & control versions/collaboration with our project.

[Due March 1st 2013]

Features:

1. GUI Interface
2. Upload & Download Capabilities
3. Connection Management
4. Connect to a Web Host (Server)
5. Browse Files
6. Add Directory
7. Delete Directory
8. Delete File
9. Change File Permissions
10. Real-time File Transfer Display

Specifications:

- We'll most likely be doing the majority of our work with C++ in Microsoft Visual Studio.
